# 104_crawler_new  

# Step1 - install  

Windows CMD（or Terminal of the VS Code）enter the following command to install the required modules.  

`pip install -r requirements.txt`

# Step2 - edit the config.txt file  

Enter the conditions you want to use for the **104 Job Bank** crawler into config.txt and save it.  


# Step3 - execute 104_crawler_new.py  

Use Python to execute 104_crawler_new.py then the jobs as you need would be saved in the 104_jobs.xlsx.
